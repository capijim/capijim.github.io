## Overview

7TCC is a comprehensive sports fashion e-commerce platform developed by a team of **8 students from Saigon University**. The platform specializes in high-quality sports apparel and accessories, featuring AI-powered customer support and multiple payment options.

## Key Features

- **Product Management**: Full catalog with categories, search, filtering, and real-time inventory tracking.
- **AI Chatbot**: Google Gemini AI integration for intelligent customer support, product recommendations, and Vietnamese language optimization.
- **Payment Integration**: MoMo e-wallet, bank transfer, credit/debit cards (Visa, JCB, ATM), and COD.
- **Admin Dashboard**: Analytics, product CRUD, order processing, user management, blog posts, and sales reports.
- **Customer Features**: Shopping cart, wishlist, product reviews, order history, email notifications.
- **Modern UI**: Responsive design with Bootstrap 5, dark/light theme, smooth animations.

## Tech Stack

- **Backend**: PHP 8.1+, MySQL 8.0+, Composer
- **Frontend**: HTML5, CSS3, JavaScript ES6+, Bootstrap 5, jQuery
- **AI**: Google Gemini API
- **Payment**: MoMo Payment Gateway
- **Email**: PHPMailer
- **Deployment**: Heroku, AWS RDS
