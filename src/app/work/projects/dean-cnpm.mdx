## Overview

DeAnCNPM is an e-commerce web application for clothing sales, developed as a capstone project for the Software Engineering course. Built with **PHP** and **MySQL**, the project follows a traditional MVC-like structure with admin control panel and customer-facing storefront.

## Key Features

- **Product Catalog**: Browse and search products by categories.
- **Shopping Cart**: Add products to cart and checkout.
- **User Authentication**: Customer registration and login system.
- **Admin Panel**: Manage products, orders, and news articles.
- **News Section**: Blog/news page for store updates.
- **Database Management**: SQL-based product and order management.

## Tech Stack

- **Backend**: PHP
- **Database**: MySQL
- **Frontend**: HTML, CSS, JavaScript
- **Team**: 7 contributors
