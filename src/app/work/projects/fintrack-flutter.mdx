## Overview

FinTrack is a personal finance management application developed with **Flutter**, helping users track income and expenses, manage multiple accounting books, set savings goals, and analyze spending effectively.

## Key Features

- **Book Management**: Create and manage multiple accounting books with independent balance tracking.
- **Transaction Management**: Add, edit, delete transactions with categories, detailed notes, and full history.
- **Charts & Statistics**: Pie charts for expense distribution, bar charts for income/expense trends over time.
- **Savings Goals**: Flexible savings (custom amounts) and periodic savings (daily/weekly/monthly auto-save) with progress tracking.
- **Notifications & Reminders**: Recurring bill notifications, savings goal reminders, push notifications.
- **Multi-language**: Supports Vietnamese and English with easy switching.
- **Modern UI**: Material Design 3, responsive layout, dark/light theme.

## Tech Stack

- **Framework**: Flutter 3.2.3+
- **State Management**: Riverpod
- **Database**: SQLite (sqflite)
- **Charts**: fl_chart
- **Notifications**: flutter_local_notifications
- **Responsive**: flutter_screenutil
